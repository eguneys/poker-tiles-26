# [morchess.com](https://morchess.com)


Mor Chess 3 is a chess puzzle game where the goal is to arrange the pieces so each piece attacks the same pieces as the target position.

It features daily puzzles, a public leaderboard, and a tutorial.


MorChess is written as a Single Page Application using the SolidJS framework and uses WebGL for rendering the actual game. The Leaderboards API is written in NodeJS using the express framework. All the puzzles are taken from Lichess Puzzle Database or fetched from Lichess API.


Use [Github issues](https://github.com/eguneys/poker-tiles-26/issues) for bug reports and feature requests.
